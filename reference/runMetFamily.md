# Start the web app in browser

Run the web app in browser

## Usage

``` r
runMetFamily(launch.browser = TRUE, ...)
```

## Arguments

- launch.browser:

  Launch web browser to open MetFamily

- ...:

  pass additional parameters down to runApp(), e.g. host="0.0.0.0",
  port=3838
