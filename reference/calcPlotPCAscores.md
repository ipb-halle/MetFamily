# PCA plot of samples

PCA plot of samples

## Usage

``` r
calcPlotPCAscores(
  pcaObj,
  dataList,
  filterObj,
  pcaDimensionOne = 1,
  pcaDimensionTwo = 2,
  showScoresLabels,
  xInterval = NULL,
  yInterval = NULL,
  downloadLayout = FALSE
)
```

## Arguments

- pcaObj:

  list object

- dataList:

  list object

- filterObj:

  filter obj

- pcaDimensionOne:

  first PCA axis

- pcaDimensionTwo:

  second PCA axis

- showScoresLabels:

- xInterval:

  numeric

- yInterval:

  numeric

- downloadLayout:

  boolean, set to TRUE for a different figure better suited for printing

## Value

makes a plot, returns NULL
