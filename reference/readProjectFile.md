# Read MetFamily Project data saved by the export function

Supports reading from plain and gzip'ed files

## Usage

``` r
readProjectFile(file, progress = FALSE)
```

## Arguments

- file:

  Path to file to read

- progress:

  Whether to update a shiny Progress bar

## Value

A big dataList.
