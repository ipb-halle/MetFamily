# parse MSP file to list

**\[superseded\]** Superseded with parseMSP rewrite in 2025-08

## Usage

``` r
parseMSP_chunk(
  fileLines,
  minimumIntensityOfMaximalMS2peak,
  minimumProportionOfMS2peaks,
  neutralLossesPrecursorToFragments,
  neutralLossesFragmentsToFragments,
  offset = 0,
  progress = FALSE
)
```

- fileLines:

- minimumIntensityOfMaximalMS2peak:

  numeric

- minimumProportionOfMS2peaks:

  numeric

- neutralLossesPrecursorToFragments:

  boolean

- neutralLossesFragmentsToFragments:

  boolean

- offset:

- progress:

  boolean

## Value

list
