# Analyze tree

Recursive function. FIX creates many global variables

## Usage

``` r
analyzeTree(
  dataList,
  cluster,
  filter,
  nodeIdx,
  minimumProportionOfLeafs = 0.75
)
```

## Arguments

- dataList:

  list object

- cluster:

  cluster object

- filter:

  filter

- nodeIdx:

  numeric

- minimumProportionOfLeafs:

  numeric

## Value

list object
