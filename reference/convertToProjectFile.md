# Convert to Project File

Reads and creates a list from separate files.

## Usage

``` r
convertToProjectFile(
  filePeakMatrixPath,
  fileSpectra,
  parameterSet,
  progress = FALSE
)
```

## Arguments

- filePeakMatrixPath:

  path

- fileSpectra:

  path

- parameterSet:

  list of parameters

- progress:

  logical

## Value

resultObj
